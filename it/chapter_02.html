LESSON 2 Advanced Features of Excel

Charts are an effective way to display data in a pictorial form. They make it easier to draw comparisons and analyse growth, relationships, and trends among values in a range. When you insert a chart, the data is displayed in a special window called Datasheet. The chart is updated automatically with changes in data.

COMPONENTS OF A CHART

CHART AREA — The chart area includes all objects and elements in a chart.
CATEGORY AXIS — Category axis or X-axis is the horizontal axis of a chart.
VALUE AXIS — Value axis or Y-axis is the vertical axis used to plot the values.
DATA SERIES — A data series is a related set of data values.
AXIS TITLES — These are the headings given to X-axis and Y-axis. The titles help in understanding what is being depicted on the axes.
PLOT AREA — The plot area is a window within a chart area. It contains the actual chart and includes the plotted data, data series, category, and value axis.
LEGEND — It depicts the colours, patterns, and symbols assigned to a data series.
CHART TITLE — It describes the aim and contents of the chart.
GRIDLINES — These can either be horizontal or vertical lines depending on the selected chart type. They extend across the plot area of the chart. Gridlines make it easier to read and understand the values.

  • The chart area encompasses everything visible in the chart frame.
  • The plot area is strictly inside the axes where data is plotted.
  • Axes titles label the X and Y axes for clarity.
  • Gridlines assist in reading values accurately from the chart.
  • The legend explains the meaning of different colors or symbols used.

COMMONLY USED CHART TYPES

Column Chart — A column chart is a commonly used chart type to display data in the form of vertical bars. It is used to show changes in data over a period of time or comparisons among different data items.
Bar Chart — This chart displays data in the form of long rectangular rods, also called bars, that are placed vertically or horizontally in the chart area. A bar chart illustrates comparisons amongst the individual items.
Line Chart — This chart is in the form of lines. It is very similar to plotting a graph on a graph paper with its values on the X and Y-axis. A line chart uses connecting dots to display trends in data over a period of time.
Pie Chart — It is a circular chart divided into sectors where each sector shows the relative size of each value. It always shows only one data series and is useful when you want to emphasise on a significant element.
Doughnut — This chart shows the relationship of part to a whole. It displays data in the form of rings where each ring represents a data series added into the chart. Unlike a pie chart, the doughnut chart represents more than one data series.

  • Column charts use vertical bars with categories on the horizontal axis.
  • Bar charts use horizontal bars with categories on the vertical axis.
  • Line charts are best for showing trends over time.
  • Pie charts show parts of a whole using sectors.
  • Doughnut charts are similar to pie charts but can show multiple data series.
  • Pressing F11 creates a chart instantly in a new sheet.

CREATING A CHART

Design — One of the two tabs named as Design and Format that appear under the Chart Tools category of the Ribbon after creating a chart.
Format — One of the two tabs named as Design and Format that appear under the Chart Tools category of the Ribbon after creating a chart.
Select Data — An option accessed by right-clicking on the chart to change the data range of your chart.

  • Select the data range including column headings and row labels.
  • Click on the Insert tab and select the Column drop-down menu.
  • Click on Clustered Column in the drop-down list to create the chart.
  • Use the Design tab to change chart styles and layouts.
  • Right-click and select Select Data to modify the data range later.

CHART ELEMENTS

Add Chart Element — A button in the Chart Layouts group on the Design tab used to add components like titles, labels, and legends.
Chart Title — A text box that appears to describe the aim and contents of the chart.
Axis Titles — Headings given to X-axis and Y-axis to explain what is being depicted.
Data Labels — Values displayed directly on the chart elements to show exact figures.
Data Table — A table option that displays the underlying data used for the chart below the chart area.
Gridlines — Lines extending across the plot area to help read values.

  • Click Add Chart Element to access options for titles, labels, and grids.
  • You can choose Above Chart or Centered Overlay for the Chart Title.
  • Primary Horizontal adds an X-axis title, while Primary Vertical adds a Y-axis title.
  • Data labels can be positioned in various locations around the data points.
  • The Chart Elements button (+) allows quick addition or removal of elements.

ADDING A CHART TITLE

  • Click on Add Chart Element in the Chart Layouts group on the Design tab.
  • Select Chart Title option and choose Above Chart or Centered Overlay.
  • Type the desired text in the Chart Title text box.

ADDING AXIS TITLE

  • Select Axis Titles option from the Add Chart Element button.
  • Select Primary Horizontal option to add title for an X-axis.
  • Select Primary Vertical option to add title for a Y-axis.

ADDING DATA LABELS

  • Click on Add Chart Elements button in the Chart Layouts group.
  • Click on Data Labels option from the drop-down menu.
  • Choose any option from the displayed list to position the labels.

ADDING DATA TABLE

  • Click on Add Chart Elements button in the Chart Layouts group.
  • Click on Data Table option from the drop-down menu.
  • Choose the desired option from the displayed list.

ADDING LEGEND

  • Click on Add Chart Elements button in Chart Layouts group.
  • Click on Legend option from the drop-down menu.
  • Choose any desired option from the displayed list to place the legend.

ADDING GRIDLINES

  • Click on Add Chart Elements button in Chart Layouts group.
  • Click on Gridlines option from the drop-down menu.
  • Select any desired option to show horizontal or vertical gridlines.

FORMATTING A CHART

Format Plot Area — An option accessed by right-clicking on the Plot Area to modify fill, border, and background settings.
Format Legend — An option accessed by right-clicking on the Legend boundary to modify text, background, boundary, and colour.
Format Axis — An option accessed by right-clicking on an axis to change scale, bounds, and units.
Format Data Series — An option accessed by right-clicking on the data series to change bar colours, lines, slices, and position.

  • Formatting improves the appearance by changing text, font, line colour, and width.
  • Right-click on Plot Area to open the Format Plot Area task pane.
  • Use Fill tab to select Solid fill, Gradient fill, or Picture texture.
  • Use Border tab to select Solid line and adjust width and color.
  • You can change the background of the Chart Area similarly using Format Chart Area.

CHANGING BACKGROUNDS, COLOURS, AND PATTERNS

  • Right-click on the Plot Area and select Format Plot Area option.
  • Click on the Fill tab and select the Solid fill radio button.
  • Select the Color button and choose the desired colour from the Fill Color drop-down menu.
  • Click on the Border tab to select Solid line and increase the width.

FORMATTING THE LEGEND

  • Right-click on the Legend boundary and select Format Legend option.
  • Select the Fill & Line tab and then select the Border option.
  • Specify the Width and select the desired colour from the Color drop-down menu.
  • Select the Legend Options tab to position the legend (e.g., Top).

CHANGING THE SCALE OF VALUE AXIS

  • Point the mouse at Y-axis and right-click to select Format Axis option.
  • Select the Axis Options tab in the Format Axis task pane.
  • Enter Minimum and Maximum values in the respective text boxes.
  • Similarly, we can change the scale for X-axis.

FORMATTING DATA SERIES

  • Right-click on the data series and select Format Data Series option.
  • Click on Fill & Line to change border type and width.
  • Select Dash type and Width options to customize the appearance.
  • You can also choose Picture or Texture fill for the data series.

COMBO CHART

Combo Chart — A combination of two or more chart types in a single chart to plot two or more sets of values.
Secondary Axis — An option used in Combo Charts to allow different scales for different data series.

  • Enter the data required for the combo chart.
  • Go to Insert tab and select Insert Combo Chart drop-down menu.
  • Select Create Custom Combo Chart from the options.
  • Select Clustered Column for one series and Line chart for another.
  • Check the Secondary Axis box for the series requiring a different scale.

SPARKLINES

Sparklines — Mini charts that fit into a single worksheet cell to provide the visual representation of the data.
Line — A type of sparkline that represents the relative value of the selected worksheet data.
Column — A type of sparkline where the selected worksheet data is represented by tiny columns.
Win/Loss — A type of sparkline where selected worksheet data appears as a win/loss chart.

  • Sparklines are used to analyse trends in your data within a single cell.
  • Once added, further options can be accessed through the Sparklines tab.
  • Wins are represented by blue squares that appear above red squares in Win/Loss.
  • Select the data without column headings before inserting sparklines.
  • Specify the Location Range where the sparklines graphics should be created.

USING GOAL SEEK

Goal Seek — A feature that helps in determining input values needed to achieve a specific goal.
What-if Analysis — A method performed by Goal Seek to calculate a value based on a given set of values.

  • Goal Seek calculates a value by performing What-if Analysis on a given set of values.
  • Select the Data tab and click on the What-If Analysis button.
  • In the Set cell text box, define the cell address containing the formula.
  • Type the target value in the To value text box.
  • In the By changing cell text box, point to the input value you want to adjust.

GROUPING WORKSHEETS

Grouping Worksheets — A feature that facilitates in editing multiple worksheets at the same time.
[Group] — A word that appears in the Title bar indicating that sheets are grouped together.

  • Hold down the Shift key and click on Sheet tabs to select multiple sheets.
  • Enter common data, formatting effects, and formulas in the grouped sheets.
  • The word [Group] appears in the Title bar when sheets are grouped.
  • To ungroup, click on any Sheet tab other than the grouped ones.
  • You cannot delete all the sheets in a workbook; at least one must exist.

CONSOLIDATING DATA

Consolidating Data — Helps in combining the data of multiple worksheets.
Consolidate — An option in the Data Tools group used to combine data from various ranges.

  • Consolidation combines data of various ranges in the same sheet or different sheets.
  • It is essential to group the sheets together to consolidate data in different sheets.
  • Select the Sum function from the list box in the Consolidate dialog box.
  • Click on the Reference text box and select the cell range from the first sheet.
  • Click Add to include the reference in the All references text box.

SORTING DATA

Sorting Data — Means arranging the data either in an ascending or descending order.
Sort — A button in the Sort & Filter group used to arrange data.
Sort & Filter — A group on the Data tab containing sorting and filtering options.

  • Data can be sorted in rows on the basis of text, numbers, or dates.
  • Select the cell range and click on the Sort button in the Data tab.
  • Select the field on the basis of which you want to sort the data.
  • Select the My data has headers checkbox to exclude the first row.
  • You can sort more than one column by clicking the Add Level button.

FILTERING DATA

Filter — A feature that helps you to display only those records that meet the specified criteria and hide rest of the records.
Filter Button — A button in the Sort & Filter group used to activate filters.

  • Click on the Filter button in the Sort & Filter group on the Data tab.
  • Small arrows will be added in the column header to specify conditions.
  • Uncheck Select All and select specific values to filter the data.
  • Click Clear button to remove all filters from the worksheet.
  • Click Reapply button to filter new data after modifying existing data.

USING ADVANCED FILTER

Advanced Filter — An option used to filter the data in multiple fields using specified criteria, to copy the filtered record to a different location, or to find unique records.
Criteria Range — The range that must have column labels and criteria values for filtering.
List range — The range of the data that needs to be filtered.

  • Insert blank rows above the range to use as a criteria range.
  • The criteria range must have column labels and no blank rows between values.
  • Select the Data tab and click on the Advanced button in Sort & Filter group.
  • Select Copy to another location radio button in the Action section.
  • Define the Criteria range and the target location in the Copy to box.

Question: What is the difference between chart area and plot area?

Answer:
* Chart Area: The chart area includes all objects and elements in a chart. It is the outer boundary of the entire chart.
* Plot Area: The plot area is a window within the chart area. It contains the actual chart and includes the plotted data, data series, category, and value axis.

Chart is an effective way to display data in a pictorial form. Charts make it easier to draw comparison and analyse the growth, relationship, and trends among the values in a range. Charts provide more accurate analysis of information. When you insert a chart, the data is displayed in a special window called Datasheet. Chart is updated automatically with the change in data.

COMPONENTS OF A CHART

Following are the various components of a chart:

  1. CHART AREA: The chart area includes all objects and elements in a chart.
  2. CATEGORY AXIS: Category axis or X-axis is the horizontal axis of a chart.
  3. VALUE AXIS: Value axis or Y-axis is the vertical axis used to plot the values.
  4. DATA SERIES: A data series is a related set of data values.
  5. AXIS TITLES: These are the headings given to X-axis and Y-axis. The titles help in understanding what is being depicted on the axes.
  6. PLOT AREA: The plot area is a window within a chart area. It contains the actual chart and includes the plotted data, data series, category, and value axis.
  7. LEGEND: It depicts the colours, patterns, and symbols assigned to a data series.
  8. CHART TITLE: It describes the aim and contents of the chart.
  9. GRIDLINES: These can either be horizontal or vertical lines depending on the selected chart type. They extend across the plot area of the chart. Gridlines make it easier to read and understand the values.

COMMONLY USED CHART TYPES

Different types of charts are available in Excel. Some of the most commonly used charts are:
[Icon of Column Chart] Column Chart COLUMN: A column chart is a commonly used chart type to display data in the form of vertical bars. It is used to show changes in data over a period of time or comparisons among different data items. In this type of chart, categories are represented on the horizontal axis and values along the vertical axis.
[Icon of Bar Chart] Bar Chart BAR: This chart displays data in the form of long rectangular rods, also called bars, that are placed vertically or horizontally in the chart area. A bar chart illustrates comparisons amongst the individual items. In this type of chart, categories are represented on the vertical axis and values are represented on the horizontal axis.
[Icon of Line Chart] Line Chart LINE: This chart is in the form of lines. It is very similar to plotting a graph on a graph paper with its values on the X and Y-axis. A line chart uses connecting dots to display trends in data over a period of time.
[Icon of Pie Chart] Pie Chart PIE: It is a circular chart divided into sectors where each sector shows the relative size of each value. It always shows only one data series and is useful when you want to emphasise on a significant element.
[Icon of Doughnut Chart] Doughnut DOUGHNUT: This chart shows the relationship of part to a whole. It displays data in the form of rings where each ring represents a data series added into the chart. Unlike a pie chart, the doughnut chart represents more than one data series.

  1. Select any cell within the database range, say D1. Click on the Filter button in the Sort & Filter group on the Data tab.

  1. Click on the arrow next to the Hindi column header in cell D1. Uncheck the Select All check box to deselect all the check boxes. Now select the check boxes of the values 82 and 98 from the drop-down list. Click OK.

USING ADVANCED FILTER

Advanced Filter option is used to filter the data in multiple fields using specified criteria, to copy the filtered record to a different location, or to find unique records. Let us use this option.

  1. Insert at least three blank rows above the range, which is to be used as a criteria range, by clicking on the Home tab and selecting the Insert > Insert Sheet Rows option.

  2. The criteria range must have column labels. There should not be any blank row between the criteria values and the range.

  3. Select the range A4:D4. Press Ctrl+C to copy the range.

  4. Select the cell A1 and press Ctrl+V. Type the criteria which you want to match in the row below the column labels. Type = 10000 under the field name Salary i.e., D2.

  5. Click on a cell within the data range. Select Data tab and click on the Advanced button [icon] in the Sort & Filter group.
