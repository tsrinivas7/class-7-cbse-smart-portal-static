EQUATIONS

Equality and Statement of Equality

  • Equality — A mathematical expression which involves the sign of equality (= sign).
  • Statement of Equality — An expression such as 8 - 5 = 3, 6 + 3 = 9, or 9 * 5 = 45.
  • Equation — A statement of equality which involves variables.
  • An equation must involve the equality sign; signs like greater than (>) or smaller than (<) do not make it an equation.
  • An equation remains the same when the LHS and RHS are interchanged.

Components of an Equation

  • Variable — An unknown quantity whose value varies from one problem to another.
  • Left Hand Side (LHS) — The part of the equation to the left of the equality sign.
  • Right Hand Side (RHS) — The part of the equation to the right of the equality sign.
  • In general, an equation is a condition on a variable stating that LHS and RHS should have equal value.
  • Example: In y + 5 = 13, (y + 5) is the LHS and 13 is the RHS.

Types of Equations

  • Linear Equation — An equation in which the highest power of all variables is one.
  • Simple Linear Equation — An equation in which there is only one variable with its highest power as one.
  • Examples of linear equations: x + y = 9, 3a - b = 67.
  • Examples of simple linear equations: x - 4 = 13, x/4 = 14.
  • Equations can have one or more variables and each variable can have a power of one or more.

Balancing an Equation

  • An equation is similar to a weighing balance where equal weights are placed on both pans.
  • Quantities of the same value are needed to be placed on both the pans of the weighing balance.
  • If we want to perform some operation on an equation, this operation has to be performed on both sides.
  • To maintain equality, whatever is done to one side must be done to the other side.
  • This rule applies to addition, subtraction, multiplication, and division.

Root and Solution of an Equation

  • Root — The value of the variable which satisfies an equation.
  • Solution — Another term for root; the value of the variable which makes LHS equal to RHS.
  • There is only one root or solution of a simple linear equation.
  • To verify a solution, substitute the value into the equation and check if LHS equals RHS.
  • Example: In 3x - 4 = 11, substituting x = 5 gives LHS = 3 * 5 - 4 = 11 = RHS. Thus, x = 5 is the root.

SOLVING AN EQUATION

  • Solving an equation means to find its root or solution.
  • The goal is to find the value of the variable which will make the LHS equal to the RHS.
  • Methods to solve an equation include Trial and Error, Systematic, and Transposition methods.

Trial and Error Method

  • Trial and Error Method — A method where different values are substituted for the variable till LHS becomes equal to RHS.
  • Substitute different values for the variable until the correct root is found.
  • This value of the variable is known as the root or solution of the equation.
  • Applicable when the equation is very simple and the root is a small integer.
  • If the root is a fraction, it becomes impossible to guess using this method.

Systematic Method by Separating the Variable

  • Systematic Method — A method based on the rules of balancing an equation to separate the variable.
  • Perform opposite operations on both the sides to isolate the variable.
  • We may have to carry out more than one mathematical operation to separate the variable.
  • When multiplying one side by a non-zero number, multiply the other side also by the same number.
  • When dividing one side by a number, divide the other side also by the same number.

Transposing Method

  • Transposition Method — A method involving shifting of a number or variable from one side to the other side of the equality sign.
  • Transpose — Changing side is called transposing.
  • Transposing a number or expression is same as adding/subtracting/multiplying/dividing it on both the sides.
  • In doing so, the sign of the number or expression has to be changed.
  • Rules: '+' becomes '-', '-' becomes '+', '×' becomes '÷', '÷' becomes '×'.

SOLUTION TO EQUATION

  • We can form an equation from a given solution.
  • When solving an equation, we follow steps; similarly, when a solution is given, we follow steps to get the equation.
  • Start with the solution and apply operations to both sides to frame the equation.
  • Many simple equations can be framed from a given solution.
  • Given an equation, there is a unique solution but given a solution, there are many equations.

APPLICATIONS OF SIMPLE EQUATIONS

  • Equations are used to solve day-to-day life problems.
  • Follow specific steps while solving word problems of linear equations.
  • Read the word problem carefully and find what is given and what is to be found out.
  • Analyse and understand the relationship between the known quantities and the unknown quantity.
  • Denote the unknown quantity by a variable, frame an equation, and solve it.

Wrap Up

  • A simple linear equation is an equation that involves only one variable with highest power 1.
  • The value of a variable for which an equation is satisfied is called the root/solution of the equation.
  • In trial and error method, the variable is substituted with different values to find that particular value for which the value of LHS becomes equal to the value of RHS.
  • While solving an equation using the method of balancing an equation, we can add or subtract the same number from both the sides.
  • We can multiply or divide both the sides of the equation by the same non-zero number.
  • In transposition method, numbers or expressions are shifted/transposed from one side of the equation to the other side by changing their sign.

  • Every simple linear equation has exactly one unique solution.

  • You can add or subtract the same number from both sides of an equation.

  • Many different equations can be framed from a single given solution.

Key Terms

Balancing an Equation — The principle that any operation performed on one side of an equation must also be performed on the other side to maintain equality.
Equality Sign — The symbol used to indicate that two expressions have equal value.
Equation — A mathematical statement of equality involving variables and an equality sign.
Frame an Equation — The process of creating an algebraic equation from given conditions or known solutions.
Left Hand Side (LHS) — The part of the equation located to the left of the equality sign.
Linear Equation — An equation in which the highest power of all variables is one.
Perimeter — The total length of the boundary of a closed geometric figure.
Quotient — The result obtained when one number is divided by another.
Right Hand Side (RHS) — The part of the equation located to the right of the equality sign.
Root — The value of the variable which satisfies an equation making LHS equal to RHS.
Simple Linear Equation — An equation containing only one variable with its highest power as one.
Solution — The value of the variable which satisfies an equation making LHS equal to RHS.
Statement of Equality — A mathematical expression that involves the sign of equality.
Systematic Method — A method of solving an equation by separating the variable using opposite operations on both sides.
Transposition Method — A method involving shifting a number or variable from one side of the equation to the other by changing its sign.
Trial and Error Method — A method of solving an equation by substituting different values for the variable until LHS equals RHS.
Variable — An unknown quantity whose value varies from one problem to another.
Word Problems — Real-life problems that can be solved using algebraic equations.

TUNE UP ACTIVITY

Question: Work in pairs and take a weighing balance and some unit cubes. Place 12 unit cubes on right hand side and 5 unit cubes on the left hand side of the weighing balance. Now, try to balance the two sides of weighing balance by adding unit cubes to the side with only 5 unit cubes. Let ( x ) unit cubes will be added on the left hand side for balancing. Can this situation of weighing balance be represented by the equation ( x + 5 = 12 )? Try to find the value of ( x ) by balancing the two sides of this weighing balance.

Answer:
Yes, the situation can be represented by the equation ( x + 5 = 12 ).
To balance the weighing balance, the weight on both sides must be equal.
Right Hand Side (RHS) = 12 unit cubes.
Left Hand Side (LHS) = 5 unit cubes + ( x ) unit cubes.
Equation: ( x + 5 = 12 ).
Subtracting 5 from both sides:
( x = 12 - 5 )
( x = 7 )
So, 7 unit cubes need to be added to the left hand side to balance the weighing balance.

Question: Solve by the method of balancing an equation and write the steps used to separate the variable.
a) ( x + 7 = 10 )
b) ( 3x + 2 = 8 )

Answer:
a) ( x + 7 = 10 )
Step 1: Subtract 7 from both sides to isolate x.
( x + 7 - 7 = 10 - 7 )
( x = 3 )

b) ( 3x + 2 = 8 )

( 3x + 2 - 2 = 8 - 2 )
( 3x = 6 )

( 3x / 3 = 6 / 3 )
( x = 2 )

Question 4

Question: Solve by the transposition method.
a) ( 6(t - 2) = 18 )
b) ( 3/2 p - 4 = 20 )
c) ( x/2 + 3/4 = 7/8 )
d) ( 2(x + 4) - 10 = 44 )
e) ( (4 - x)/6 = 7 )
f) ( 2y + 7/2 = 39/2 )

Answer:
a) ( 6(t - 2) = 18 )
Transpose 6 to RHS (divide):
( t - 2 = 18 / 6 )
( t - 2 = 3 )
Transpose -2 to RHS (add):
( t = 3 + 2 )
( t = 5 )

b) ( 3/2 p - 4 = 20 )
Transpose -4 to RHS (add):
( 3/2 p = 20 + 4 )
( 3/2 p = 24 )
Transpose 3/2 to RHS (multiply by reciprocal 2/3):
( p = 24 \times 2 / 3 )
( p = 16 )

c) ( x/2 + 3/4 = 7/8 )
Transpose 3/4 to RHS (subtract):
( x/2 = 7/8 - 3/4 )
( x/2 = 7/8 - 6/8 )
( x/2 = 1/8 )
Transpose 2 to RHS (multiply):
( x = 1/8 \times 2 )
( x = 1/4 )

d) ( 2(x + 4) - 10 = 44 )
Transpose -10 to RHS (add):
( 2(x + 4) = 44 + 10 )
( 2(x + 4) = 54 )
Transpose 2 to RHS (divide):
( x + 4 = 54 / 2 )
( x + 4 = 27 )
Transpose 4 to RHS (subtract):
( x = 27 - 4 )
( x = 23 )

e) ( (4 - x)/6 = 7 )
Transpose 6 to RHS (multiply):
( 4 - x = 7 \times 6 )
( 4 - x = 42 )
Transpose 4 to RHS (subtract):
( -x = 42 - 4 )
( -x = 38 )
Multiply by -1:
( x = -38 )

f) ( 2y + 7/2 = 39/2 )
Transpose 7/2 to RHS (subtract):
( 2y = 39/2 - 7/2 )
( 2y = 32/2 )
( 2y = 16 )
Transpose 2 to RHS (divide):
( y = 16 / 2 )
( y = 8 )

### EQUATIONS

While solving arithmetic problems, we often come across mathematical expressions such as \( 8 - 5 = 3 \), \( 6 + 3 = 9 \) and \( 9 \times 5 = 45 \).

Such mathematical expressions which involve the sign of equality, \( = \) sign, are called statements of equality or equality.

A statement of equality, which involves variables, is called an equation.

[IMAGE_START] Diagram of a weighing balance with 12 unit cubes on the right side and 5 unit cubes on the left side. The balance is not yet balanced. [IMAGE_END]

<!-- Page: 95 -->

For example:
$x + 9 = 12$
$4m = 20$
$4y + 5 = 6y - 25$
$\frac{t}{2} + 5 = 11$

All the above examples represent equations, as they comprise two things, namely a variable (unknown quantity whose value varies from one problem to another) and an equality sign.

The part of the equation to the left of the equality sign is called the Left Hand Side (LHS) and the part to the right of the equality sign is called the Right Hand Side (RHS) of the equation. In general, an equation is a condition on a variable. The condition states that the expression on LHS and the expression on RHS should have equal value.

For example, in the equation $y + 5 = 13$, $(y + 5)$ is the LHS and 13 is the RHS.

Equations can have one or more variables and each variable can have a power of one or more. The equation in which the highest power of all variables is one is called a linear equation. The equation in which there is only one variable with its highest power as one is called a linear equation in one variable or simple linear equation. For example, $x + y = 9$, $3a - b = 67$, etc. are all linear equations but not simple linear equations. On the other hand, $x - 4 = 13$, $\frac{x}{4} = 14$, etc. are simple linear equations.

Grasp It
- An equation must involve the equality sign. If there is some other sign, such as greater than $(>)$ or smaller than $(<)$, it is not an equation.
- An equation remains the same when the LHS and RHS are interchanged.

Solved Examples

Example 1: Express each of the following statements as an equation.
a) The sum of twice a number $y$ and 15 is 27.
b) Three-fourths of a number $l$ minus 8 gives 4.

Solution:
a) Twice a number $y$ is $2y$. Sum of $2y$ and 15 can be written as $2y + 15$.
Thus, the required equation is $2y + 15 = 27$.
b) Three-fourths of a number $l$ is $\frac{3}{4}l$. $\frac{3}{4}l$ minus 8 can be written as $\frac{3}{4}l - 8$.
Thus, the required equation is $\frac{3}{4}l - 8 = 4$.

Example 2: Express each of the following equations as a statement.
a) $2t + 5 = 9$
b) $\frac{k}{5} = 4$

Solution:
a) 5 added to twice of a number $t$ gives 9.
b) A number $k$ when divided by 5 gives 4 as a quotient.

Balancing an Equation

In the tune-up section, we have seen that an equation is similar to a weighing balance in which equal weights are placed on both the pans so that the arm of the weighing balance remains exactly in the balanced position. Quantities of the same value are needed to be placed on both the pans of the weighing balance.

Similarly, this rule applies to equations also. If we want to perform some operation on an equation, this operation has to be performed on both the

the variable which satisfies an equation is called

## SOLVING AN EQUATION

Solving an equation means to find its root or
solution, that is, the value of the variable which
will make the LHS equal to the RHS of the
equation. The methods to solve an equation are
Trial and Error method, Systematic method and
Transposition method.

### Trial and Error Method

In this method, we substitute different values for
the variable till we find the value for which the
value of LHS becomes equal to the RHS of the
equation. This value of the variable is known as
the root or solution of the equation. For example,
let us solve the equation 2y - 5 = 7 by the trial
and error method.
Now, we substitute different values for y to find
the solution of the equation.

| Value of y | LHS (2y - 5) | RHS | Conclusion |
|------------|---------------|-----|-------------|
| 1          | 2 × 1 - 5 = -3 | 7   | -3 ≠ 7      |
| 2          | 2 × 2 - 5 = -1 | 7   | -1 ≠ 7      |
| 3          | 2 × 3 - 5 = 1  | 7   | 1 ≠ 7       |
| 4          | 2 × 4 - 5 = 3  | 7   | 3 ≠ 7       |
| 5          | 2 × 5 - 5 = 5  | 7   | 5 ≠ 7       |
| 6          | 2 × 6 - 5 = 7  | 7   | 7 = 7       |

Since the LHS and RHS are equal when y = 6;
therefore, 6 is the solution of 2y - 5 = 7.

The trial and error method is only applicable
when the equation is very simple and the root
is a small integer. If the root is a fraction, it
becomes impossible to guess.

## Systematic Method by
## Separating the Variable

This method of solving an equation is based
on the rules of balancing an equation which
we have learnt in the previous section. We
can use these rules to separate the variable by
performing opposite operations on both the
sides. In our attempt to separate the variable
in an equation, we may have to carry out more
than one mathematical operation.

### Solved Examples

#### Example 3: Solve \(\frac{x}{4} = -4\).

Solution: Here, x is divided by 4. So, to
separate x, we will multiply \(\frac{x}{4}\) by 4. But,
when we multiply one side of an equation
by a non-zero number, we must multiply the
other side also by the same number.
So, \(\frac{x}{4} \times 4 = -4 \times 4\)
x = -16

#### Example 4: Solve 4y + 7 = 19 and verify
your answer.

Solution: 4y + 7 = 19
Subtracting 7 from both the sides, we get
4y + 7 - 7 = 19 - 7
4y = 12
Dividing both the sides by 4, we get
\(\frac{4y}{4} = \frac{12}{4}\)
y = 3

We can easily verify whether \( y = 3 \) is the solution of the equation \( 4y + 7 = 19 \) or not. Substituting \( y = 3 \) in the given equation, we get LHS \( = 4 \times 3 + 7 = 12 + 7 = 19 = \) RHS. Thus, \( y = 3 \) is the solution of the equation \( 4y + 7 = 19 \).

Transposing Method

This method involves shifting of a number or variable from one side to the other side of the equality (=) sign by changing its sign without affecting the equality of the equation.

Grasp It

Changing side is called transposing. Transposing a number or expression is same as adding/subtracting/multiplying/dividing it on both the sides. In doing so, the sign of the number or expression has to be changed. \( '+' \) becomes \( '-' \) \( ' \times ' \) becomes \( ' \div ' \) \( '-' \) becomes \( '+' \) \( ' \div ' \) becomes \( ' \times ' \)

Solved Examples

Example 5: Solve \( 12x + 1 = 25 \) using the transposition method.

Solution: \( 12x + 1 = 25 \)

\(\Rightarrow 12x = 25 - 1 \) (Transposing 1 from LHS to RHS and reversing the operation)

\(\Rightarrow 12x = 24 \)

\(\Rightarrow x = \frac{24}{12} \) (Transposing 12 from LHS to RHS and reversing the operation)

\(\therefore x = 2 \)

Example 6: Solve \( 3(m - 5) = 21 \)

Solution: \( 3(m - 5) = 21 \)

\(\Rightarrow (m - 5) = \frac{21}{3} = 7 \)

\(\therefore m = 7 + 5 = 12 \)

Example 7: Solve \( \frac{y}{5} + 3 = 2 \)

Solution: \( \frac{y}{5} + 3 = 2 \)

\(\Rightarrow \frac{y}{5} = 2 - 3 = -1 \)

\(\therefore y = -1 \times 5 = -5 \)

SOLUTION TO EQUATION

We have learnt how to solve a given equation. Can we form an equation from a given solution? Yes, we can. When we solve an equation, we follow some steps. Similarly, when a solution of an equation is given, we need to follow some steps to get the equation. For example, let us frame an equation for the solution \( x = 2 \).

Start with \( x = 2 \).

Multiplying both the sides by 3, we get

\( 3x = 2 \times 3 = 6 \)

Adding 11 on both the sides, we get

\( 3x + 11 = 6 + 11 = 17 \)

Can we frame another equation for the solution \( x = 2 \)? Let us try to do that.

Start with \( x = 2 \).

Adding 4 on both the sides, we get

\( x + 4 = 2 + 4 = 6 \)

Multiplying both the sides by 5, we get

\( 5(x + 4) = 6 \times 5 = 30 \)

So \( 5(x + 4) = 30 \) is another equation for which the solution is \( x = 2 \).

Thus, we can frame many equations from a given solution.

Grasp It

Given an equation, there is a unique solution but given a solution, there are many equations.

#### Example 8
There are some balls in box A and the number of balls in box B is 30 more than those in box A. If the total number of balls is 150, find the number of balls in each box.
Solution: Let the number of balls in box A be $x$.
Number of balls in box B = $x + 30$
Total number of balls = $(x + x + 30)$
But the total number of balls is given to be 150.
So, $x + x + 30 = 150$
$\Rightarrow 2x + 30 = 150$
$\Rightarrow 2x = 150 - 30 = 120$
$\Rightarrow x = \frac{120}{2} = 60$
Thus, the number of balls in box A and B is 60 and 90, respectively.

# Wrap Up

- A simple linear equation is an equation that involves only one variable with highest power 1.
- The value of a variable for which an equation is satisfied is called the root/solution of the equation.
- In trial and error method, the variable is substituted with different values to find that particular value for which the value of LHS becomes equal to the value of RHS in the equation.
- While solving an equation using the method of balancing an equation, we can
  - add or subtract the same number from both the sides of the equation.
  - multiply or divide both the sides of the equation by the same non-zero number.
- In transposition method, numbers or expressions are shifted/transposed from one side of the equation to the other side by changing their sign.
- Many simple equations can be framed from a given solution.
